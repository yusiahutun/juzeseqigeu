How to Play Cyberpunk 2077 Phantom Liberty in VR for Free

June 26, 2024
blog

Quick Links

  • Play Cyberpunk 2077 Phantom Liberty in VR Using the VorpX mod
  • Play Cyberpunk 2077 in VR Using the R.E.A.L VR Mod

Key Takeaways

  • To play Cyberpunk 2077 in VR for free, you need to download the VorpX VR mod from its website, then set it up based on the game's install location and the kind of VR headset you use.
  • Alternatively, you can use the R.E.A.L VR mod by Luke Ross, which provides native VR support for Cyberpunk 2077. However, it is not available for free.
  • You might also need to revert the game back to its older version if you are using the R.E.A.L VR mod.

Cyberpunk 2077 had one of the most disastrous game launches in recent memory. However, CD Projekt Red have redeemed themselves with the newest Phantom Liberty expansion. This is the perfect chance to revisit Night City and totally immerse yourself in the game with a VR headset.

Play Cyberpunk 2077 Phantom Liberty in VR Using the VorpX mod

Download the Cyberpunk 2077 VR mod from the VorpX website. Once downloaded, extract the file and run CyberpunkVR_Setup.exe.

The mod will then ask you to locate the Cyberpunk 2077 game folder. If you don't know the install location of the game you can find it by following the steps below.

  • Steam: Steam Library > Right-click on Cyberpunk 2077 > Manage > Browse Local Files
  • GOG: GOG Library > Right-click on Cyberpunk 2077 > Manage Installation > Show Folder
  • Epic Games: Library > Cyberpunk 2077 > Three-dot menu > Manage > Press the file icon in front of Installation.

Continue though the prompts. The mod will install the free version of VorpX for Cyberpunk 2077 on your PC. At the end of the installation enable "Start Cyberpunk VR Config" and select finish.

In the config menu, select the software you will be using to connect your VR headset to the PC.

Connect your VR headset to your PC. Launch the game and enjoy Cyberpunk 2077 Phantom Liberty in VR for free.

If you want to use Steam VR , make sure Steam Overlay and Desktop Game Theater are disabled in Steam General Settings before launching the game.

Play Cyberpunk 2077 in VR Using the R.E.A.L VR Mod

Luke Ross' R.E.A.L VR mod is the closest thing to native VR support for Cyberpunk 2077. Unfortunately, you will have to sign up to Luke Ross's Patreon account, which costs $10 per month, to get full access to the mod. However, you can also use the mod to play numerous other games in VR such as Elden Ring, Horizon Forbidden West, and more.

At the time of writing this article, the mod only works with version 1.63 of the game. Ross has promised to fix the mod, but it might take a while. Until then, you can try out the VR mod by reverting the game to an older version by following the steps below.

Installing the R.E.A.L VR mod is relatively simple. Additionally, the installation process will likely be the same when it's updated to work with Cyberpunk 2077 version 2.0. Once the mod has been updated, you can just follow the steps below to install the R.E.A.L VR mod without reverting the game to version 1.63.

How to Revert Cyberpunk 2077 Back to Version 1.63

If you're using Steam, open your Steam library. Right-click on Cyberpunk 2077 and select Properties. Go to BETAS and select "1_63_legacy_patch". The game will automatically update and revert to version 1.63. If you want to return to the newest version of the game, simply follow the same steps and select "None" from the Betas menu.

If you're using GOG, then go to the “Installed” section. Select Cyberpunk 2077 and go to Manage Installation > Configure. Change Beta Channels to "1_63_legacy_patch" and disable the auto-updates option. After choosing your preferred version, select OK. Unfortunately, if you have bought the game on Epic Games Store, then you won't be able to revert it to an older version.

How to Install R.E.A.L VR Mod for Cyberpunk 2077

Sign up for Luke Ross’s official Patreon page, then download the latest version of the R.E.A.L VR mod and extract the zip file.

Move the contents of the file to the same folder as Cyberpunk2077.exe. The default location will look similar to this:

C:Program Files(x86)\Steam\SteamLibrary\steamapps\common\Cyberpunk 2077\bin\x64

Go to the location where you extracted the files (Cyberpunk 2077\bin\x64). Find and run the “RealConfig.bat” file and let it set up the VR mod.

Connect your PC to a VR headset and launch the game. Thanks to the R.E.A.L VR mod, Cyberpunk 2077 will now launch in VR mode. Enjoy.
